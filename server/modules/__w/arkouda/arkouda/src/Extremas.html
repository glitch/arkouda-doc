Extremas¶

Usage

use Extremas;

or

import Extremas;
record KExtreme¶

KExtreme data structure that is used to track the minimum or maximum size values that are in an array. Acts similar to a heap data structure, but can be sorted and merged to extract the extreme values from multiple kextreme objects.

This record is used in the user-defined kreduce reuction by having one :record: KExtreme instance per task, which is then built up by pushing the values into the heap-like :record: KExtreme by the accumulate() function. Once all of the values from each task have been accumulated, the combine() function is called on each :record: KExtreme instance, which is handled by sorting the record, giving up its heap-like quality, and using it as a sorted array to increase the efficiency of the merge step.
